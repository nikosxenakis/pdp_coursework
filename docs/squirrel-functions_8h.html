void initialiseRNG ( long *  seed)

Initialises the random number generator, call it once at the start of the program on each process. The input value should be negative, non-zero and different on each process, I suggest something like -1-rank where rank is the MPI rank. ran2 is a flawed RNG and must be used with particular care in parallel however it is perfectly sufficient for this course work The seed is modified and then passed to each other subsequent function that relies on the random generator, which also modify the seed
